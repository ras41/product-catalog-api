# Product Catalog API

A backend application built with Node.js, Express.js, and PostgreSQL for browsing a large product catalog with efficient cursor-based pagination.

Live - `https://product-catalog-api-2mh4.onrender.com`

## Features

* Cursor-based pagination
* Category filtering
* Batch seeding of 200,000 products
* Simple product browsing UI

## Tech Stack

* Node.js
* Express.js
* PostgreSQL (Supabase)

## Run Locally

```bash
npm install
node scripts/seed.js
node server.js
```

Open: `http://localhost:3000`
